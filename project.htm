Welcome to me projects page.

The current project is called "Mario v2.0".

It is a modification to the existing game "Mario & Luigi" created by Mike Wiering.

It originally had six levels, but now it will now have about 23.

 

I have ended all development on this game, due to the lack of a level editor, so making levels is a time consuming process. Unless I find a useable level editor, I will not make any more changes to this game.

 

Here is a list of added features:

-17 more levels

 

-2 new graphics

 

-2 new ground/wall graphics. Both were created by the original author of this game but for some reason or another, he did not use them. I did not have to add any code to make these work, since they were ready to go, they were not used in the original levels.

 

-Time limit feature. Unfortunately the code that makes the time limit work isn't very good. So it will be about 3 minutes on ALL levels

 

-Moving platform. This tile already exists in the game, but was never used. So I didn't have to add code.

 

-New level name layout. Instead of ”Level 5" it now displays "World 2-1".

 The game starts at World 1 and advances to the next world every 4 levels. ( like the original Mario games did)

 

-Rearranged status bar. It now looks more like the original status layout of Super Mario Bros. 1.

Incase you haven’t figured out the controls of the game yet, here they are:

CTRL+Arrowkeys=RUN

ALT=JUMP

ALT+CTRL=Long Jump

SPACE=Shoot fireballs

ARROW KEYS=Walk

LShift=Scroll Screen to the left.

RShift=Scroll Screen to the right.
For more info run the game in DOS with the following command switch: Mario -?
